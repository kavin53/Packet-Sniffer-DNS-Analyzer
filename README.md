# Python Packet Sniffer & DNS Analyzer

## Overview
This project is a simple network packet sniffer built in Python using the Scapy library.  
The tool captures network packets in real time and analyzes basic packet information such as:

- Source IP
- Destination IP
- Protocol type
- DNS queries

This project was built as a cybersecurity learning exercise to understand how network traffic monitoring works.

---

## Features
- Real-time packet sniffing
- Protocol detection (TCP / UDP)
- Source and destination IP analysis
- DNS query detection
- Terminal-based packet output

---

## Technologies Used
- Python
- Scapy
- Npcap (Windows packet capture driver)
